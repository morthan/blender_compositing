Свойства вида
^^^^^^^^^^^^^

Окно свойств вида вызывается путём выбора из меню 3D-вида пункта *View* -> *View
Properties...*. Оно позволяет настроить параметры окна 3D-вида.

image:img/figure2_16.png[]

*Рисунок 2.16* 'Окно свойств вида'

Группа контролов *Grid* отвечает за настройки координатной сетки на экране:

Spacing::
    Расстояние между линиями координатной сетки. По умолчанию равно 1 единице
    Blender.

Lines::
    Количество линий сетки в случае перспективного отображения.

Divisions::
    На сколько частей делится сторона квадрата более высокого уровня для
    получения квадрата низшего уровня. Для наглядной демонстрации увеличьте
    масштаб 3D-вида, прокручивая колёсико мыши. Обратите внимание: при
    увеличении внутри каждого большого квадрата строится сетка из более мелких.
    По умолчанию в этом поле стоит число 10, так что сторона маленького квадрата
    будет равна 0.1 единицы Blender.

В группе *View Camera* находятся настройки камеры 3D-видаfootnote:[Не путайте
камеру 3D-вида с объектом камеры! Камера 3D-вида отрисовывает изображение
редактируемой сцены в окне 3D-вида, а камера-объект используется для рендеринга
готовой картинки.]:

Lens::
    Угол зрения камеры в перспективном режиме.

Clip Start::
    Минимальное расстояние от объекта до камеры, начиная с которого объекты
    отрисовываются. (Объекты, расположенные слишком близко к перспективной
    камере, подвергаются сильным искажениям и могут испортить картинку.)

Clip End::
    Максимальное расстояние от объекта до камеры, на котором объекты ещё
    отрисовываются. Используется чтобы экономить вычислительные ресурсы, не
    отрисовывая слишком далеко расположенные объекты.

Группа кнопок *Display* отвечает за общие настройки отображения объектов в
3D-виде:

Outline Selected::
    Если эта кнопка нажата, контуры выделенного объекта обводятся другим цветом.

All Object Centers::
    Центр объекта отрисовывается маленьким кружочком. Если эта кнопка выключена
    (по умолчанию), то он отображается только для выбранного в данный момент
    объекта. Если кнопку включить -- для всех объектов сцены.

Relationship Lines::
    Кнопка включает отображение линий взаимосвязиfootnote:[Про них, в частности,
    про связь «родитель-потомок», будет рассказываться немного позже.]. По
    умолчанию она включена, так что если в вашей сцене один объект зависит от
    другого, то между ними будет протянута тонкая пунктирная линия.

Solid Tex::
    Обычно, для ускорения отрисовки, в «сплошном» ('solid') режиме грани
    заливаются однотонной краской, примерно соответствующей цвету материала. Но,
    если включить эту кнопку, то на гранях будет отображаться текстура,
    соответствующая материалу.

Группа *3D Display* отвечает за настройки отображения самого 3D-вида в
трёхмерном режиме (допустим, при виде из камеры, или если покрутить сцену
зажатой средней кнопкой мыши):

Grid Floor::
    Включает/выключает отображение горизонтальной плоскости «пола» в нулевых
    координатах.

X Axis, Y Axis, Z Axis::
    Включают/выключают отображение координатных осей X (красная линия), Y
    (зелёная) и Z (синяя).

С помощью полей группы *3D Cursor* можно точно задать координаты 3D-курсора по
X, Y и Z в единицах Blender.

Поля группы *View Locking* позволяют зафиксировать виртуальную камеру 3D-вида на
указанном объекте (для этого нужно ввести имя объекта в поле *Object*) или кости
арматуры (поле *Bone*). После этого как бы вы ни вращали 3D-вид, выбранный
объект/кость неизменно будет оставаться в центре.

Кнопки группы *Keyframe Display* включают отображение ключевых кадровfootnote:[К
сожалению, эти настройки, судя по всему, относятся только к анимированным
материалам и/или арматуре. По крайней мере, изменение этих настроек для ключей
простого перемещения объекта никаких изменений поведения Blender не вызвало.]:

Active::
    Включает отображение ключевых кадров только для активного в данный момент
    объекта.

Muted::
    Включает отображение ключевых кадров для выключенных в данный момент
    каналов.

Local::
    Включает отображение только тех ключевых кадров, которые привязаны
    непосредственно к блоку данных объекта.

Material::
    Включает отображение ключевых кадров, назначенным материалам.

ShapeKey::
    Включает отображение ключевых кадров, назначенным ключам деформации.
