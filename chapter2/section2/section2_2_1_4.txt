File Paths
^^^^^^^^^^

Здесь расположены пути ко всяким интересным для Blender местам:

YFexport::
    Путь к папке, куда по умолчанию экспортируется XML для Yafray. Если вы
    пользуетесь фотореалистичным рендером Yafray, то этот путь надо прописывать.
    Мы будем пользоваться встроенным рендером, так что нам это не нужно.

Fonts::
    Путь к папке, содержащей шрифты.

Textures::
    Путь к папке, где по умолчанию ищутся текстуры.

Tex Plugins::
    Путь к папке, в которой по умолчанию ищутся текстурные плагины.

Seq Plugins::
    Путь к папке, в которой по умолчанию ищутся плагины видеоредактора.

Render::
    Путь к папке, куда по умолчанию будет рендериться результат вашей работы.

Python Scripts::
    Путь к папке с питоновскими скриптами.

Sounds::
    Путь к папке, в которой по умолчанию ищутся звуковые файлы.

Temp::
    Путь ко временной папке.

[NOTE]
На самом деле прописывать эти пути совершенно не обязательно. В 99% случаев их
можно указать при выполнении нужной операции (рендеринг, добавление текстуры и
т.д.). Просто если у вас уже есть устоявшаяся структура папок и вы точно знаете,
где что лежит, то прописывание этих путей сэкономит вам несколько секунд
времени.
